# Mathematical & Utility Tools
A collection of Python scripts designed to automate common calculations and unit conversions.

## Featured Tools:
* **Compound Interest Calculator:** A tool for modeling investment growth over time using core financial formulas.
* **Metric Converter:** Accurate conversion utility between different measurement systems.
* **Countdown Timer:** A clean, terminal-based productivity tool using the `time` module.
